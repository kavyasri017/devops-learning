# 📘 Server & Cloud – Simple Notes

## 1️⃣ Storage Types

* **HDD** → Slow, cheap → Backup காக use பண்ணுவாங்க
* **SSD** → Fast → OS & normal servers
* **NVMe** → Very fast, costly → High performance servers

✅ Production server → SSD / NVMe
✅ Backup storage → HDD

---

## 2️⃣ VM (Virtual Machine)

* Oru physical server la many VMs create பண்ணலாம்
* Each VM = Separate server மாதிரி work பண்ணும்
* VM backup மிகவும் important

---

## 3️⃣ AWS (Cloud)

Amazon Web Services

AWS provides:

* Server
* Storage
* Networking
* Backup

---

## 4️⃣ Server vs Cloud

### 🖥 Normal Server

* Physical machine
* Public IP use பண்ணி connect
* Hardware cost அதிகம்

### ☁ Cloud Server

* Internet மூலம் access
* No hardware tension
* Easy to scale
* **Pay As You Go** (use பண்ணின அளவுக்கு தான் money)

---

## 5️⃣ Pay As You Go

* 1 hour server run → 1 hour charge
* 10GB storage → 10GB charge

---

## 6️⃣ Backup Types

* Full Backup → Full data copy
* Incremental → Changes மட்டும்
* Differential → Last full backup பிறகு changes

---

## 7️⃣ S3 Storage

Amazon S3

* Backup storage
* File storage
* Highly secure & scalable

---

## 8️⃣ ESXi

VMware ESXi

* Physical server ல VM create பண்ண use பண்ணுவாங்க

---

## 9️⃣ VPC (Networking)

Amazon VPC

* Cloud உள்ள private network
* Secure environment

---

# 🔟 Important Points

* VM backup முக்கியம்
* HDD → Backup
* SSD/NVMe → Production
* Cloud → Pay only for what you use
